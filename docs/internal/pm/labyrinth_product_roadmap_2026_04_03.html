OPEN SEC-011: TOCTOU Race in LoreDocs Export
LoreDocs storage.py file export checks dest.exists() then writes -- classic TOCTOU race. Low urgency on single-user machine but must be fixed before any multi-user deployment. Remediation: atomic file creation with os.open(dest, os.O_CREAT | os.O_EXCL) or tempfile + rename.
Severity: MEDIUM (single-user, low urgency) | Owner: Ron | Priority: Before multi-user launch
OPEN MEG-036: SQL Optimizer Test Debt
Two SQL credits concurrency tests (test_credits_concurrency.py) are failing against current defensive implementation. Tests were written for older behavior that raised exceptions; implementation now recovers gracefully instead. Tests need updating. Accumulating test debt.
